# `.override-log.md`

Append-only audit log of `OVERRIDE:` actions taken in this repo, per
CLAUDE.md §1.3. Entries are added in the same commit that implements
the override action, and the commit subject begins `[OVERRIDE]`. The
commit-msg hook installed by `scripts/install-hooks.sh` enforces both
invariants when `SAWYER_OVERRIDE=1` is set.

Format:

```
- YYYY-MM-DD HH:MM UTC | operator=<git user.email> | action=<brief summary> | reason=<justification> | commits=<sha,sha or "self">
```
